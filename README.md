# crawlTaobao

# 爬取天猫店商品评论信息



## 爬取思路

先用crawlerTaobaoItem爬取商品列表。接着根据商品id爬取商品的第一页评论，解析出商品以及对应的评论页数。再根据商品以及页数爬取对应的评论。
